GitHub Wiki Search Engine Enablement is a tool to allow GitHub Wikis to be indexed by search engines. There's no robots.txt on this site. Because of this, text content served from this tool should be indexed.

Currently, GitHub prevents crawling of repository's Wiki pages. This means Wikis are invisible to Google and other search engines such as DuckDuckGo. This can be a very unpleasant surprise if you're looking to share knowledge or put a FAQ on a Wiki!

This tool simply mirrors the rendered contents of the pages and has a link at the top to visit the page on GitHub for normal usage. The content of the tool is not meant to be read by users.

Usage

  1. Go to this site at /mirror/{account name}/{repo}/wiki and get a link.
  2. For best results, place the link in a location that can be indexed by search engines.

Examples

  • nelsonjchen/github-wiki-test
  • commaai/openpilot
